# Transcription Conventions

## Document Metadata

- **Title**: 18.785 Number Theory I, Lecture #1: Absolute Values and Discrete Valuations
- **Author**: Andrew V. Sutherland
- **Institution**: MIT (Fall 2021)
- **Subject**: Algebraic number theory — absolute values, discrete valuations, integral extensions

## Math Notation

- Inline math: `$...$`
- Display math: `$$...$$`
- Use standard LaTeX commands (e.g., `\mathbb{Z}`, `\lvert \cdot \rvert`, `\coloneqq`)

## Environments

- **Definition N.N**: Transcribe as `**Definition N.N.**` followed by the content
- **Theorem N.N**: Transcribe as `**Theorem N.N.**` followed by the statement
- **Proposition N.N**: Transcribe as `**Proposition N.N.**` followed by the statement
- **Corollary N.N**: Transcribe as `**Corollary N.N.**` followed by the statement
- **Lemma N.N**: Transcribe as `**Lemma N.N.**` followed by the statement
- **Proof**: Transcribe as `*Proof.* ...` ending with `$\square$`
- **Remark N.N**: Transcribe as `**Remark N.N.**` followed by the content
- **Example N.N**: Transcribe as `**Example N.N.**` followed by the content

## Numbered Items

Use the numbering from the source document exactly. Do not renumber.

## Lists and Enumerations

- Numbered lists within proofs or definitions: use `(i)`, `(ii)`, `(iii)` etc. as in the source
- Lettered items: use `(a)`, `(b)`, `(c)` etc. as in the source

## References

- Cross-references to other items: use the original numbering (e.g., "by Theorem 1.3")
- External references: transcribe bibliographic references as they appear (e.g., `[1, Thm. 3.2]`)

## Page Boundaries

Each page is transcribed into its own file (`pages/N.md`). If a theorem, proof, or other environment spans a page boundary, the continuation is noted at the top of the next page with a comment like `<!-- continued from page N -->`.
